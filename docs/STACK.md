# Stack Tecnológico - Electiva 5

Documento para comunicar al docente el stack utilizado.

---

## Stack Definido

| Categoría | Tecnología | Versión / Notas |
|-----------|------------|-----------------|
| **Frontend** | HTML5 | Estructura semántica |
| **Estilos** | CSS3 | Variables, Grid, Flexbox |
| **Lógica** | JavaScript (ES6+) | Vanilla, sin frameworks |
| **Base de datos** | IndexedDB | Persistencia local en navegador |
| **Control de versiones** | Git | Con estrategia de ramas |
| **Repositorio remoto** | GitHub | Pull Requests hacia dev y main |
| **Gestión de proyecto** | Jira | Backlog, sprints |
| **Documentación** | Notion | Roles, decisiones, notas |

## Justificación

- **IndexedDB:** No requiere backend; los datos persisten localmente. Adecuado para prototipos y desplegar en GitHub Pages.
- **Vanilla JS:** Menor complejidad, sin dependencias externas.
- **Git + ramas:** Cumple con el estándar dev-nombre-apellido y flujo de PR.

## Requisitos del Sistema

- Navegador moderno (Chrome, Firefox, Edge, Safari)
- No se requiere instalación adicional
