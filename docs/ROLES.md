# Definición de Roles - Electiva 5

Plantilla para copiar a Notion. Define los roles del equipo de desarrollo.

---

## Roles del Equipo

| Rol | Responsabilidades | Integrante |
|-----|-------------------|------------|
| **Scrum Master** | Facilitar ceremonias, eliminar impedimentos, seguimiento del sprint | |
| **Product Owner** | Priorizar backlog, definir criterios de aceptación, validar entregables | |
| **Tech Lead** | Revisar código, definir estándares técnicos, guiar decisiones técnicas | |
| **Desarrollador Frontend** | Maquetación HTML/CSS, lógica JavaScript, UI/UX | |
| **Desarrollador Base de Datos** | Esquema IndexedDB, persistencia, migraciones | |
| **QA / Tester** | Pruebas funcionales, reporte de bugs, validación de criterios | |
| **Documentador** | README, documentación técnica, plantillas Notion/Jira | |
| **Integrador** | Revisar Pull Requests, merge a dev/main, resolver conflictos | |

---

## Notas

- Un integrante puede asumir más de un rol según el tamaño del equipo.
- El estándar de ramas es: `dev-nombre-apellido` (ej: dev-ana-martinez).
