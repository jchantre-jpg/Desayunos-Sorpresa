# Guía para configurar en Notion

Pasos para replicar esta documentación en Notion.

## 1. Crear workspace o página

- Crea una página principal: **"Electiva 5 - Registro de Asistencia"**
- Puedes usar una base de datos o páginas hijas

## 2. Páginas sugeridas

| Página | Contenido |
|--------|-----------|
| **Roles** | Copia `docs/ROLES.md` |
| **Backlog** | Copia `docs/BACKLOG.md` (puedes crear tabla) |
| **Stack Tecnológico** | Copia `docs/STACK.md` |
| **Sprint Planning** | Copia `docs/SPRINT-PLANNING.md` |
| **Decisiones** | Decisiones técnicas del equipo |
| **Links** | Enlace a GitHub, Jira |

## 3. Base de datos en Notion (opcional)

- **Tipo:** Base de datos de tabla
- **Propiedades:** ID, Historia, Prioridad, Estado, Asignado, Sprint
- Importa las user stories del `BACKLOG.md`

## 4. Compartir con el docente

- Configura el workspace/página como **"Compartir"**
- Añade al docente con permiso de **"Puede ver"** o **"Puede editar"**
- Envía el enlace en el formato: `https://www.notion.so/tu-workspace/...`
